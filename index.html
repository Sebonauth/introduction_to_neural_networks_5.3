Let's see how information actually flows through these layers.

The process follows these steps:

  1. Input values enter through the input layer
  2. Each connection carries the value forward, multiplying it by a weight
  3. Each node in the next layer sums its inputs
  4. An activation function transforms the sum
  5. The process repeats until reaching the output

Forward Propagation

The process of data flowing forward through the network, from input to output, with each layer performing its computations in sequence.

Let's express this mathematically. For any node in a hidden or output layer:

\[ z^{[l]}_j = \sum_{i=1}^{n} w^{[l]}_{ji} a^{[l-1]}_i + b^{[l]}_j \] \[ a^{[l]}_j = \sigma(z^{[l]}_j) \]

Where:

  • \(z^{[l]}_j\) is the weighted sum for node j in layer l
  • \(a^{[l]}_j\) is the activation of node j in layer l
  • \(w^{[l]}_{ji}\) is the weight connecting to node j from node i
  • \(b^{[l]}_j\) is the bias for node j
  • \(\sigma\) is the activation function
